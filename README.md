# Web Chatbot

A Python-based web chatbot project.

## Features
- Chatbot using Flask
- Stores chat history in SQLite
- Responsive UI

## Installation
1. Clone the repo: `git clone <repo-url>`
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `python app.py`
